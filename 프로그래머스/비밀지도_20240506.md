## 풀이
- 2진수에서 1인 부분이 한번만 나와도 벽이므로 OR 연산을 한다.
- 10진수를 2진수로 변환하고, 길이에 맞게 왼쪽에 0을 채워준다.
- 벽과 공백 부분에 맞는 문자로 변환한다.
```js
function solution(n, arr1, arr2) {
    return arr1
      .map((n, i) => {
          return n | arr2[i]
      })
      .map(n => n.toString(2))
      .map(str => str.padStart(n, '0'))
      .map(str => str.replaceAll('1', '#').replaceAll('0', ' '))
}
```
